# Pixora

React 19 + Vite application for GenPic Master.

## Prerequisites

- Node.js >= 22.22.0
- pnpm >= 10.0.0

## Development

```powershell
corepack pnpm@10.33.2 install
corepack pnpm@10.33.2 dev
```

## Scripts

- `pnpm build` - build the Vite app.
- `pnpm lint` - run ESLint.
- `pnpm test` - run Vitest.
